# 项目技术文档

## 技术栈

- 语言：Go，编译为单二进制 exe
- 无外部依赖（除 YAML 配置解析）
- 无数据库，预算数据纯内存缓存
- HTTP 服务：标准库 net/http

## 目录结构

```
BudgetProject/
├── config.yaml        # 运行时配置（与 exe 同目录部署）
├── config/            # 备选配置目录（优先于根目录 config.yaml）
│   └── config.yaml
├── src/
│   ├── main.go        # 入口 + 定时同步 + 路由
│   ├── config.go      # 配置加载（支持 config/ 子目录优先）
│   ├── store.go        # 内存缓存（sync.RWMutex map）
│   ├── ekb.go          # 合思 API 客户端（Token + 单据查询 + 维度追溯）
│   ├── checker.go      # 业务校验逻辑（分支规则）
│   ├── sync.go         # 预算数据同步（按白名单拉取预算包）
│   ├── handler.go      # HTTP 接口处理
│   └── logger.go       # 日志轮转（daily/weekly/monthly）
├── docs/
│   ├── 校验规则.md      # 面向业务的校验规则说明
│   └── 逻辑梳理.md      # 本文件
└── README.md
```

## 配置加载优先级

1. 命令行 `-config` 指定路径 → 直接使用
2. `config/config.yaml` → 优先
3. `config.yaml` → 兜底

## 日志

- 自动创建 `logs/` 目录
- 文件命名按轮转周期：daily=`2026-05-20.log`、weekly=`2026-W21.log`、monthly=`2026-05.log`
- 同时输出到终端和文件

## 配置项

| 配置项 | 说明 |
|--------|------|
| server.port | HTTP 端口，默认 8000 |
| ekuaibao.host | 合思 API 域名 |
| ekuaibao.app_key / app_secret | 合思开放平台凭证 |
| expense_nature | 费用性质 ID → 中文名称映射 |
| budget_targets | 预算包白名单（支持 ID 精确匹配或名称关键词匹配） |
| budget_targets[].depth | 预算包钻取深度 |
| sync.interval_minutes | 自动同步间隔（分钟） |
| logging.rotation | 日志轮转周期 daily/weekly/monthly |

## 单据字段提取

从合思 API `/v1.1/flowDetails/byCode` 获取单据详情，提取 4 个字段：

| 字段 | 合思字段路径 | 说明 |
|------|------------|------|
| 成本中心 | form.E_system_costcenter | 成本中心 ID |
| 费用档案 | form.details[0].feeTypeForm.u_费用类型档案 → form.u_费用类型档案 | 优先从明细取，兜底从表单取 |
| 项目 | form.项目 | 项目 ID |
| 费用性质 | form.u_费用性质 → 配置映射 | ID 映射为 业务/管理/生产/未知 |
